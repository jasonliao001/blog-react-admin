## 项目介绍

基于 create-react-app 搭建的 React 项目脚手架。

## 启动

```
npm install
npm run mock
npm start
```

## Dependencies

- React 16.7 --hooks
- Redux --状态管理
- React-Router-V4 --多页面
- Ant Design --UI
- React Intl --国际化
- Gulp --合并国际化词条
- Restify --Mock 服务器
